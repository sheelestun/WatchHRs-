# WatchHRs Desktop Client

Десктопное приложение для мониторинга активности сотрудников.

## Основные функции
- **Авторизация по лицу**: Вход в систему с помощью веб-камеры через сервис распознавания.
- **Мониторинг активности**: Подсчет кликов мыши и нажатий клавиш клавиатуры.
- **Автоматические скриншоты**: Снятие скриншота рабочего стола каждые 10 минут.
- **Интеграция с API**: Отправка статистики и скриншотов на центральный сервер.

## Установка

1. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Запуск**:
   ```bash
   python main.py
   ```

## Тестовый режим
Для отладки без подключения к бэкенду предусмотрен тестовый режим:
```bash
python main.py --test
```
В этом режиме:
- Авторизация всегда проходит успешно (через 1 сек).
- Статистика и скриншоты не могут никуда отправиться.
- Интервал снятия скриншотов ускорен (установлен на 10 секунд для удобства тестирования).

## Сборка EXE (Windows)

Для создания исполняемых файлов выполните следующие шаги:

1. **Установите pyinstaller** (если еще не установлен):
   ```bash
   pip install pyinstaller
   ```

2. **Запустите файл сборки**:
   - В CMD/PowerShell:
     ```bash
     build.bat
     ```
   - В Git Bash:
     ```bash
     ./build.bat
     ```

После завершения в папке `dist/` появятся два файла:
- `WatchHRs.exe` — Основной режим работы.
- `WatchHRs_Test.exe` — Тестовый режим (авто-авторизация, ускоренные скриншоты).

## Структура проекта
- `main.py` — Точка входа в приложение.
- `main_test.py` — Точка входа для тестового режима.
- `build.bat` — Скрипт для сборки EXE файлов.
- `core/` — Логика (мониторинг, камера, клиент API).
- `ui/` — Графический интерфейс на Tkinter.
- `screenshots/` — Локальное хранилище временных скриншотов.
